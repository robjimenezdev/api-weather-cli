# 🌤 API Weather CLI

Consulta la previsión del tiempo a 5 días desde la terminal.

## Requisitos
- Python 3.x
- Cuenta en [OpenWeatherMap](https://openweathermap.org/api)

## Instalación
1. Clona el repositorio
git clone https://github.com/robjimenezdev/api-weather-cli.git

2. Instala las dependencias
pip install -r requirements.txt

3. Copia el archivo de ejemplo
cp .env.example .env

4. Añade tu API Key en el archivo `.env`
OPENWEATHER_API_KEY=tu_clave_aqui

## Uso
python Predicción_tiempo.py

## Ejemplo
Introduce el nombre de la ciudad: Madrid

## Tecnologías
- requests
- python-dotenv
- OpenWeatherMap API
